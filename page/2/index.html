在snap中调用mediapipe的案例

‌MediaPipe‌是由Google Research开发的一款开源框架，旨在帮助开发者轻松地构建、测试和部署复杂的多模态、多任务的机器学习模型，特别擅长于实时处理和分析音频、视频等多媒体数据‌
应用场景和示例
1‌姿态估计‌：MediaPipe可以实时检测和追踪人体的关键点（如肩膀、肘部、膝盖等），并估计人体的姿态，适用于体育训练、动作捕捉、增强现实等应用‌。
2‌手部追踪‌：能够检测和追踪手部的关键点，提供手势识别和手部动作分析的能力，适用于手势控制、虚拟现实、手写输入等应用‌。
3‌人脸检测‌：用于识别图像或视频中的人脸，适用于安全监控、人脸识别登录等场景‌。
4‌物体检测‌：可以识别和检测图像或视频中的物体，适用于各种视觉相关的应用‌。
5‌运动追踪‌：提供运动追踪功能，适用于需要分析人体或动物运动的应用‌。

mediapipe 提供了 JavaScript API, 可以通过 JavaScript function 将 mediapipe 引入到 Snap! 中。
import() 可以将 ECMAScript 模块异步动态加载到非模块环境中。
在https://snap.codelab.club/可以导入 ,相关的mediapipe 库（人脸检测、手部检测、姿态检测）。在snap中使用mediapipe库，不需要修改底层代码，只需要用拖拽程序块的方式，就能体验，非常适合中小学生使用。早期尝试过在anaconda终使用mediapipe进行教学，后来又把他移植到xedu环境终，发现学生学习效果都不能达到自己要求，所以在snap中尝试一下。

snap简介--构建你自己的块

snap简介

Snap！（以前称为 BYOB）是 Scratch (https://scratch.mit.edu) 的扩展重新实现，允许您构建自己的块。非常适合中学生或大学生学习计算机科学。它是在 Smalltalk 先驱者艾伦·凯 (Alan Kay)、丹·英格尔斯 (Dan Ingalls) 思想的指引下加强了对个人计算和面向对象编程（OOP）。其 Morphic 图形框架是 Snap! 的核心。
“Logo 是伪装成 BASIC 的 Lisp”，Snap！是伪装成Scratch的方案。
Snap！是一种可以直接告诉计算机你想要它做什么的编程语言，Snap! 不是使用键盘编写程序，而是使用 鼠标拖拽的方式实现线程，我们可以把它看成是一种伪代码，而这种方式又与生成式人工智能的提示词有些类似，或许可以把它作为一种过渡阶段。
打开浏览器窗口并连接到 https://snap.berkeley.edu/run可以运行 Snap!，Snap！中国国内网站 https://snap.codelab.club 是由种瓜维护的。

研究的目的

scratch 图形化编程适合编程学习的入门，进一步的学习国内采用的是python代码教学，是不是可以尝试用图形化的方式继续进行计算机科学和人工智能的教学。通过snap尝试把十一学校郑老师所提的六个人工智能实验完成一下：
实验1：以二维表为载体，旨在让中学生认识和分析数据表的实验，数据来源可以是他们真实的物理、化学、生物课堂。
实验2：实现传统机器学习的实验，如线性回归或者鸢尾花分类的实验，用来体会机器学习的一般流程。
实验3：计算机视觉方向的基础实验，如提取图片特征等，简单跟学生讲讲怎么使用卷积提取图片边界。
实验4：计算机视觉方向的进阶实验，如基于CIFAR10这个数据集，使用卷积神经网络实现图像分类。
实验5：自然语言处理的基础实验，如直接分析一篇小说，重点让学生体会如何定量化地描述“文字”，并如何提取文字中的信息。
实验6：自然语言处理的进阶实验，如使用Pytorch框架和LSTM实现文本的分类等。
